## 🎯 方案1：使用Vercel + GitHub（推荐）

这是最简单的自动部署方案！

### 步骤：

1. **访问Vercel：** https://vercel.com/

2. **用GitHub登录**
   - 点击 "Sign Up" 或 "Login"
   - 选择 "Continue with GitHub"
   - 授权Vercel访问你的GitHub

3. **导入项目**
   - 点击 "Add New..." → "Project"
   - 找到你的仓库：`qinyiruqin-prog/GYYG`
   - 点击 "Import"

4. **配置项目**
   - Project Name: `yangyangji-v3` （或其他名称）
   - Framework Preset: `Vite`
   - Build Command: `npm run build`
   - Output Directory: `dist`
   - 点击 "Deploy"

5. **完成！** 🎉
   - 等待2-3分钟部署完成
   - 获得永久网址：`https://yangyangji-v3.vercel.app`

### 🔄 自动更新原理

**以后每次我帮你更新代码后：**

```bash
cd ~/Downloads/GYYG-improved
git add .
git commit -m "更新内容"
git push
```

**Vercel会自动：**
- ✅ 检测到代码更新
- ✅ 自动构建新版本
- ✅ 自动部署到网站
- ✅ 网址不变，内容已更新

**你不需要手动上传任何文件！**
