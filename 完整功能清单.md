## 🚀 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/qinyiruqin-prog/GYYG.git
cd GYYG
```

### 2. 安装依赖

```bash
npm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

### 4. 构建生产版本

```bash
npm run build
```

## 📝 使用指南

### 设置角色自动档

1. 打开**设置**应用（⚙️）
2. 找到**高级功能** → **角色自动档**
3. 选择角色，开启自动档
4. 设置行为间隔和行为类型

### 设置自动刷新

1. 打开**设置**应用
2. 找到**高级功能** → **自动刷新**
3. 开启自动刷新
4. 设置刷新间隔

### 手动刷新

- 桌面右上角🔄按钮
- 点击立即刷新所有角色动态
- 无论是否开启自动刷新，都可以使用
