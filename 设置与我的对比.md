# "设置" vs "我的" 功能对比

## "我的" (MeApp) 包含的功能：

### 账号区域
1. ✅ 我的账号 → 身份管理
2. ✅ API配置
3. ✅ API预设
4. ✅ 主题设置

### 功能区域
1. ✅ 分区管理
2. ✅ 角色管理
3. ✅ 本地资源（音乐/图片）
4. ✅ 数据备份（导出/导入）
5. ✅ 自动生成NPC (开关)
6. ✅ 角色主动互动 (开关 + 手动/自动)
7. ✅ 主动消息频率 (多/中/少/关)
8. ✅ 关于

---

## "设置" (SettingsScreenV3) 包含的功能：

### 基础设置
1. ⚙️ API配置 → **与"我的"重复**
2. ⚙️ API预设 → **与"我的"重复**
3. ⚙️ 主题设置 → **与"我的"重复**
4. ⚙️ 我的账号 → **与"我的"重复**

### 实验功能
1. ❌ 自动模式 (点击无反应)
2. ❌ 离线模式 (开关)
3. ❌ 向量记忆 (开关)
4. ❌ 自动刷新 (开关 + 时间间隔)

### 聊天设置
1. ✅ 内心想法透明度
2. ✅ 自动保存角色图片

### 交互设置
1. ✅ 侧边滑动返回
2. ✅ 悬浮球 (开关)

### 数据管理
1. ⚙️ 导出数据 → **与"我的"重复**
2. ⚙️ 导入数据 → **与"我的"重复**

---

## 📊 结论

### 重复功能 (应删除)
1. API配置
2. API预设
3. 主题设置
4. 我的账号
5. 导出数据
6. 导入数据

### 无效功能 (应删除或修复)
1. 自动模式 - 点击无反应
2. 离线模式 - 无实际作用
3. 向量记忆 - 无实际作用
4. 自动刷新 - 无实际作用

### 有效功能 (应合并到"我的")
1. 内心想法透明度
2. 自动保存角色图片
3. 侧边滑动返回
4. 悬浮球开关

---

## ✅ 合并方案

**保留"我的"，删除"设置"**

将"设置"中的4个有效功能合并到"我的"中：
- 在"我的"中添加"显示设置"区域
  - 内心想法透明度
  - 自动保存角色图片
  - 侧边滑动返回
  - 悬浮球开关
