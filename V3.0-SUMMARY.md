## 🎯 核心亮点

### 1. 人设工坊革命性升级

**三大分离：**
- **通用人设**（50K）：完整的角色背景、性格、经历
- **世界书**（50K）：独立的世界观设定、背景知识
- **Chat人设**（20K）：专门的聊天风格、说话习惯

**为什么分离？**
- 通用人设用于AI理解角色整体
- 世界书在触发时注入上下文
- Chat人设覆盖聊天场景，不影响其他

**字数对比：**
- v2.0：persona字段，无上限但建议<5K
- v3.0：总共120K字符，分类清晰

### 2. 小号系统

**无限创建：**
- User可创建无限小号
- Character可创建无限小号
- 每个小号独立人设

**使用场景：**
- 用户：匿名探索、角色扮演、不同身份
- 角色：测试用户、隐藏接触、多重身份

**层级显示：**
- 主号显示小号数量
- 点击展开查看所有小号
- 小号可独立编辑

### 3. 完整导入/导出

**导入功能：**
- 支持单个/批量导入
- 自动处理ID冲突
- 包含人设、世界书、Chat人设
- 错误提示友好

**导出功能：**
- 导出单个身份
- 导出所有身份
- JSON格式标准
- 可重复导入

**JSON格式示例：**
```json
{
  "nickname": "张三",
  "signature": "热爱生活",
  "persona": "我是...(最多50000字)",
  "worldbook": "世界观...(最多50000字)",
  "chatPersona": "聊天时...(最多20000字)",
  "imagePromptTemplate": "25岁男性",
  "onlineName": "网名",
  "remark": "备注",
  "isAlt": false
}
```

### 4. 心声透明度控制

**调节方式：**
- 滑块控制（0-100%）
- 实时预览效果
- 三档提示

**应用场景：**
- 角色内心想法显示
- 可根据喜好调整清晰度
- 不影响其他UI元素

## 📝 使用指南

### 如何使用新功能

#### 1. 使用增强版身份管理

```typescript
// 在你的代码中导入
import { IdentityScreenV3 } from './apps/IdentityScreenV3';

// 替换原来的 IdentityScreen
<IdentityScreenV3
  users={users}
  activeUserId={activeUserId}
  onSave={handleSave}
  onDelete={handleDelete}
  onSetActive={handleSetActive}
  onBack={handleBack}
/>
```

#### 2. 配置v3.0设置

```typescript
import { SettingsScreenV3 } from './apps/SettingsScreenV3';

<SettingsScreenV3
  innerThoughtOpacity={0.7}
  swipeBackEnabled={true}
  floatingBallEnabled={false}
  autoSaveCharImages={true}
  onUpdateSetting={(key, value) => {
    // 更新设置
  }}
  onBack={handleBack}
/>
```

#### 3. 创建小号

1. 进入用户身份管理
2. 找到主账号
3. 点击"+小号"按钮
4. 自动创建小号
5. 点击编辑自定义

#### 4. 导入完整人设

1. 准备JSON文件（参考格式）
2. 点击右上角上传图标
3. 选择JSON文件
4. 自动导入并验证

#### 5. 调节心声透明度

1. 进入v3.0功能设置
2. 找到"心声透明度"
3. 拖动滑块调节
4. 查看实时预览
