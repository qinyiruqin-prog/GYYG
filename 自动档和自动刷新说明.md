# 🤖 角色自动档和自动刷新系统 - 完整说明

## ✅ 已完成功能

### 1. 🤖 角色自动档系统

角色可以主动使用所有功能，真正"活"起来！

#### 核心功能
- ✅ **主动发消息** - 角色会主动给你发聊天消息
- ✅ **主动发朋友圈** - 角色会发布朋友圈动态
- ✅ **主动发图片** - 角色会发送图片消息
- ✅ **使用社交平台** - 角色会发微博、Twitter等
- ✅ **使用其他应用** - 角色会使用外卖、购物等功能

#### 配置选项
- **开关控制** - 每个角色独立开关
- **行为间隔** - 10分钟～6小时可调（默认30分钟）
- **行为类型** - 5种行为独立开关
- **AI生成内容** - 根据角色人设生成真实内容

---

### 2. 🔄 自动刷新系统

**两种刷新方式，完全灵活：**

#### A. 自动刷新
- ✅ 开启后自动定时刷新
- ✅ 间隔可调：1分钟～30分钟
- ✅ 后台自动运行
- ✅ 所有开启自动档的角色会执行行为

#### B. 手动刷新
- ✅ **始终可用**（无论是否开启自动刷新）
- ✅ 点击按钮立即刷新
- ✅ 所有开启自动档的角色立即执行行为
- ✅ 刷新按钮在桌面右上角（🔄图标）

**逻辑：**
```
开启自动刷新 = 自动定时刷新 + 可以手动刷新
未开启自动刷新 = 只能手动刷新
```

## 📱 使用方法

### 第1步：设置角色自动档

1. 打开**设置**应用
2. 找到**高级功能** → **角色自动档**
3. 选择要设置的角色
4. 开启**自动档开关**
5. 调整**行为间隔**（10分钟～6小时）
6. 选择**行为类型**：
   - ✅ 主动发消息
   - ✅ 主动发朋友圈
   - ✅ 主动发图片
   - ✅ 使用社交平台
   - ✅ 使用其他应用

### 第2步：设置自动刷新（可选）

1. 打开**设置**应用
2. 找到**高级功能** → **自动刷新**
3. 开启**自动刷新开关**
4. 调整**刷新间隔**（1～30分钟）

### 第3步：开始使用

**自动刷新模式：**
- 系统每隔设定时间自动刷新
- 所有开启自动档的角色会执行行为
- 你也可以随时点击刷新按钮手动刷新

**纯手动模式：**
- 不开启自动刷新
- 只在你点击刷新按钮时才刷新
- 所有开启自动档的角色会执行行为

### 核心逻辑

```typescript
// 1. 检查角色是否该行动
function shouldCharacterAct(char: Character, now: number): boolean {
  if (!char.autoMode) return false;
  
  const interval = char.autoModeInterval * 60 * 1000;
  const lastAction = char.lastAutoActionTime || 0;
  
  return now - lastAction >= interval;
}

// 2. 决定行为类型（随机）
function decideAction(char: Character) {
  const enabled = [];
  if (char.autoActions.sendMessage) enabled.push('message');
  if (char.autoActions.sendMoment) enabled.push('moment');
  // ...
  
  return enabled[Math.random() * enabled.length];
}

// 3. AI生成内容
async function generateAutoMessage(char: Character) {
  const prompt = `你是${char.name}，根据人设主动发消息...`;
  const response = await callChatRich([{role: 'user', content: prompt}]);
  return response.text;
}

// 4. 执行所有角色行为
async function processAllAutoActions(settings: AppSettings) {
  const results = [];
  
  for (const char of settings.characters) {
    if (shouldCharacterAct(char, Date.now())) {
      const action = decideAction(char);
      const result = await executeAutoAction(char, action);
      results.push(result);
      
      char.lastAutoActionTime = Date.now();
    }
  }
  
  return results;
}
```
