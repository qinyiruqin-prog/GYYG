### 2. 📱 线上/线下模式（类型系统已完成）

**核心概念：**
- **线上模式** = 微信聊天（手机上的文字/语音/图片交流）
- **线下模式** = 现实见面（模拟面对面互动、约会、活动）
- **记忆互通** = 两种模式共享同一段记忆，AI知道所有发生的事

**类型定义：**
```typescript
// 交互模式类型
export type InteractionMode = 'online' | 'offline';

// ChatMessage 添加mode字段
export interface ChatMessage {
  // ... 原有字段
  mode?: InteractionMode; // 消息的交互模式
}

// ChatThread 添加字段
export interface ChatThread {
  // ... 原有字段
  interactionMode?: InteractionMode; // 当前会话的交互模式
  sharedMemory: ChatMessage[]; // 线上线下共享的记忆
}
```

**使用场景：**
1. **线上聊天** - 用户在"聊天"应用中和角色发微信
2. **切换线下** - 点击"线下模式"应用，记录和角色的现实活动
3. **记忆互通** - AI在线上聊天时会记得线下发生的事，反之亦然

**示例：**
- 线下模式：「今天和小明去了咖啡馆，聊了两个小时」
- 切回线上模式，发微信：「今天很开心~」
- AI回复：「我也是！咖啡馆那个蛋糕真的很好吃」（记得线下的事）
